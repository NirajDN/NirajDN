## 🚀 Projects

- [🔗 OS.Event-logger)](https://github.com/NirajDN/RTOS-EVENT-LOGGER) — 🔐 Real-Time OS Security Event Logger
A real-time monitoring system that logs and visualizes OS-level security events to help detect potential vulnerabilities. It features a dynamic dashboard with WebSocket integration, enabling seamless live updates from the backend. Built with React on the frontend and Node.js/Express on the backend, it uses MongoDB for efficient event storage and retrieval. The system tracks security events in real time, providing insights into system activity for proactive monitoring and analysis.

- [🔗 Time-Explorer)](https://github.com/NirajDN/Time-Explorer)-🕰️ Time Explorer – Historical Events Finder
Time Explorer is an interactive web app that lets users discover major historical events from any selected date. If no significant event is found, it seamlessly displays top news headlines from that date, offering both history and real-world context.
